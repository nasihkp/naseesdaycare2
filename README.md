# Nasees Love & Learn Daycare & Preschool

A modern, responsive, Montessori-inspired website for **Nasees Love & Learn Daycare & Preschool** located in Mangalshery, Vellamunda.

## Features
- **Montessori-inspired design**: Warm, safe, playful, and educational colors & layout.
- **Daycare Admissions Form**: Submits direct inquiries to `nasihkp0271@gmail.com` via FormSubmit.
- **Sitemap & SEO Optimization**: Features a dynamic sitemap generator (`sitemap.xml`) and Google Site Verification.
- **Fully Responsive**: Optimized for all devices from mobile to desktop.
